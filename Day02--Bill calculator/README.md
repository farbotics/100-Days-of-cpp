# Bill Calculator 💸

A simple CLI bill calculator built in C++.

## Features
- Calculates tip amount
- Splits bill between people
- Calculates final amount per person
- Handles zero division error


## Concepts Practiced
- Variables
- User input
- Arithmetic operations
- Data types
- Output formatting
- While loop

## How to Run

Compile:
g++ main.cpp -o tipcalculator

Run:
./tipcalculator
